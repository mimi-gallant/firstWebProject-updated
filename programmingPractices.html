Good HTML Programming Practice
  • Space your code properly
    • Don't add unnecessary spaces
  • Only use one (1) h1 element per web page
  • Don't use capital letters for any names (including attributes)
    • Do this: <body>, NOT this: <BODY>
  • Always use alt text
  • Don't space your equal signs (unless you absolutely need to)
    • Do this: <img src= "image.png" alt= "a sample image" width=100 height=100/>,

      NOT this: <img src = "image.png" alt = "a sample image" width = 100 height = 100/>
  • Minimize the length of each line of code
  • Always use title element, as it is important for SEO (search engine optimization)
